# 克雷西战役 (1346年)

![战役地图示意图](/content/battles/cressy/images/01-map.jpg)

## 背景

1346年，英法百年战争爆发。英格兰国王爱德华三世率军入侵法国，在法国北部肆虐掠夺。法国国王腓力六世集结大军，发誓要将英军赶尽杀绝。

![爱德华三世指挥作战](/content/battles/cressy/images/07-kings.jpg)

英军经过长途行军，已疲惫不堪。当他们抵达克雷西附近时，法军主力已在后方紧追不舍。爱德华三世深知已无法逃脱，决定在此地与法军决一死战。

## 双方对比

| 方面 | 英格兰 | 法兰西 |
|------|--------|--------|
| 统帅 | 爱德华三世 | 腓力六世 |
| 兵力 | 约10,000人 | 约35,000人 |
| 核心武器 | 威尔士长弓 | 重骑士 |
| 战场态势 | 疲惫、以逸待劳 | 人数优势、急于复仇 |

![威尔士长弓手阵列待命](/content/battles/cressy/images/02-longbow.jpg)

## 战场布置

英军占据克雷西村北侧的高地，精心布置了防御阵地：

1. **弓箭手阵地**：将长弓手分散在阵线前沿，每个弓箭手配备充足的箭矢
2. **障碍工事**：在阵前挖掘了无数坑洞，掩盖草皮，专门破坏法军骑兵冲锋
3. **步兵掩护**：在每个弓箭手小组之间配置步兵，负责近身保护
4. **骑士预备队**：将重骑兵留在后方作为预备队

![战场全景示意](/content/battles/cressy/images/09-battlefield.jpg)

法军则盲目自信，认为凭借人数优势，一次冲锋就能粉碎英军。

## 战斗过程

### 第一阶段：轻骑兵试探

下午4点左右，法军前锋到达战场。腓力六世命令热那亚弩手（雇佣兵）先行攻击，用弩箭骚扰英军。

但热那亚弩手疲惫不堪，且当日下起了暴雨，弩弦受潮影响射程。英军长弓手则以逸待劳，箭如雨下，瞬间压制了弩手。热那亚雇佣兵仓皇后撤，被法军骑兵"督战队"砍杀，军心大乱。

![热那亚弩手被英军压制](/content/battles/cressy/images/08-crossbow.jpg)

### 第二阶段：骑士冲锋的噩梦

法军重骑士按捺不住，无视地形不利，发动了第一次冲锋。这是中世纪最壮观、也最绝望的场景：

- **泥泞阻碍**：暴雨后的战场泥泞不堪，重骑士寸步难行
- **长弓齐射**：英军长弓手开始密集齐射，箭矢如雨点般落下
- **箭雨如林**：每秒上万支箭矢射向法军，"箭矢如林"名副其实
- **骑士倒地**：法军重骑士纷纷中箭落马，盔甲再厚也挡不住近距离的箭雨

![长弓齐射箭雨如林](/content/battles/cressy/images/05-arrows.jpg)

![法军骑兵冲锋瞬间](/content/battles/cressy/images/04-charge.jpg)

### 第三阶段：十二次绝望冲锋

法军骑士们展现出了惊人的勇气，或者说愚蠢。在目睹同伴惨死后，他们仍然连续发动冲锋，总共达十二次之多：

- 每次冲锋都遭遇同样的命运：箭雨屠杀
- 尸体堆积如山，后面的骑士仍要冲锋
- 法国亲王、贵族纷纷阵亡，包括：
  - 阿朗松伯爵
  - 弗兰德斯伯爵
  - 布列塔尼伯爵
  - 洛林公爵
  - 以及10位亲王级贵族

![泥泞战场上的法军骑士](/content/battles/cressy/images/06-mud.jpg)

### 第四阶段：最后的肉搏

夜幕降临时，法军终于停止了冲锋。英军残余的骑士趁机发起反击，将疲惫的法军彻底击溃。

腓力六世在卫队拼死保护下，勉强逃出生天。战场上留下了约2000具法国重骑士的尸体，以及无数战马。

![英军庆祝胜利](/content/battles/cressy/images/10-victory.jpg)

## 战役结果

**伤亡对比：**
- 英格兰：约100人阵亡
- 法兰西：约2000-3000人阵亡（包括10位亲王级贵族）

**战略意义：**
1. **长弓威震欧洲**：证明了远程武器可以击败重骑士
2. **骑士时代终结**：重骑士在战场上的统治地位被动摇
3. **法国元气大伤**：失去了整整一代的军事精英和贵族
4. **英格兰站稳脚跟**：获得了在法国北部持续作战的基地

## 历史影响

克雷西战役成为了军事史上的经典战例，被后世反复研究：

1. **战术革命**：证明精心布置的防御阵地+远程武器可以击败人数优势的重骑兵
2. **武器变革**：长弓从此成为英军标配，在后续战役中屡建奇功
3. **社会冲击**：法国贵族大量阵亡，动摇了封建骑士制度的社会基础
4. **战争形态**：战争不再仅仅是骑士的决斗，而成为了体系化的对抗

克雷西战役之后，骑士冲锋的辉煌时代一去不复返，战争进入了新的时代。
