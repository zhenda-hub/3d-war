# 克雷西战役 - 实现文档

> **1346年 | 英格兰 vs 法兰西**
> 长弓终结重骑士时代，中世纪战争转折点

---

## 战役信息

### 基本信息
- **时间**: 1346年8月26日
- **地点**: 法国克雷西（加来以南）
- **交战方**: 英格兰(约10,000人) vs 法兰西(约35,000人)
- **伤亡**: 英格兰约100人 vs 法兰西约2,000-4,000人
- **统帅**: 爱德华三世(英) vs 菲利普六世(法)

### 战术亮点
- 威尔士长弓射程300米，每分钟10箭
- 英军防御阵地 + 坑洞阻碍骑兵
- 法军重骑士冲锋12次全部失败
- 骑士时代的终结

## 3D场景设计

### 地形

```
┌─────────────────────────────────────────┐
│  英军阵地 (高地)                         │
│  ┌─────┐ ┌─────┐ ┌─────┐ ┌─────┐      │
│  │长弓手│ │长弓手│ │长弓手│ │长弓手│      │
│  └─────┘ └─────┘ └─────┘ └─────┘      │
│  ═══坑洞障碍═════════════════════      │
│  ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓             │
│  ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓ ↓             │
│         [战场平原]                     │
│  ↑↑↑↑↑↑ 法军冲锋方向 ↑↑↑↑↑↑            │
│                                         │
│  [法军营地]                             │
└─────────────────────────────────────────┘
```

**地形要素**:
- 高地（英军阵地，约2-3米高差）
- 防御坑洞（绊马坑，约1米深）
- 平原战场（开阔草地）
- 两侧树林（限制法军展开）

### 兵种设计

#### 英格兰军
| 兵种 | 特征 | 武器 | 装备 |
|------|------|------|------|
| 长弓手 | 射程300m | 威尔士长弓(1.8-2m) | 轻甲、箭袋(24箭) |
| 重骑兵 | 少量 | 长矛、剑 | 板甲、战马 |

#### 法兰西军
| 兵种 | 特征 | 武器 | 装备 |
|------|------|------|------|
| 重骑士 | 冲锋主力 | 长枪(3-4m)、剑 | 全身板甲、盔甲马 |
| 十字弓手 | 辅助 | 十字弓 | 中甲 |
| 步兵 | 压制 | 长矛、斧 | 链甲 |

### 单位模型

#### 英格兰长弓手
- 身体比例: 8头身
- 装备细节:
  - 长弓(约2米) - 紫杉木制作
  - 箭袋 - 腰间悬挂，24支箭
  - 皮帽/无帽
  - 轻型皮甲
  - 短袍 - 红色/绿色
- 动画: 拉弓、放箭、装箭

#### 法兰西重骑士
- 身体比例: 8头身
- 装备细节:
  - 全身板甲 - 银色/金色
  - 盔甲马 - 马身披甲
  - 长枪 - 3-4米，带纹章旗
  - 骑士剑 - 腰间
  - 纹章盾 - 背面或马鞍旁
- 动画: 骑马冲锋、落马、爬起

## 战术脚本

### 阵型布局

**英军** (约300单位):
```
    [高地防御阵地]
  ┌─────────────────┐
  │ 骑     骑     骑 │  后卫
  │                 │
  │ 弓 弓 弓 弓 弓 弓 │  长弓手
  │ 弓 弓 弓 弓 弓 弓 │
  │ ══坑洞障碍═════ │
  └─────────────────┘
```

**法军** (约500单位):
```
[多次冲锋波次]
第1波: ═════════════  重骑士冲锋
第2波: ═════════════  重骑士冲锋
第3波: ═════════════  十字弓+步兵
...
第12波: 最后冲锋
```

### 战斗阶段

**阶段0: 部署 (0-5秒)**
- 英军在高地列阵，挖掘坑洞
- 法军在远处集结
- 相机扫过双方阵型

**阶段1: 第1-3次冲锋 (5-30秒)**
- 法军重骑士冲锋
- 英军长弓齐射
- 骑士在坑洞前被截停
- 箭矢穿透盔甲
- 法军伤亡惨重

**阶段2: 混乱阶段 (30-60秒)**
- 法军后续部队被阻挡
- 冲锋队形混乱
- 后续部队撞上前方溃兵
- 英军持续射击

**阶段3: 夜幕降临 (60-70秒)**
- 战场陷入黑暗
- 法军残部撤退
- 英军获胜

**阶段4: 次日清晨 (可选)**
- 清点战利品
- 法军贵族伤亡统计
- 战场上的箭矢如林

## 技术实现要点

### 箭雨系统

**箭矢弹道**:
```javascript
// 箭矢飞行轨迹
初始位置: 长弓手位置
目标位置: 骑士阵型中心
飞行时间: 约2-3秒
抛物线高度: 最高约50-100米
```

**箭矢效果**:
1. **集体齐射** - 200-300支箭同时发射
2. **弧线飞行** - 抛物线轨迹
3. **命中效果**:
   - 击中盔甲: 叮当声，火花
   - 击中战马: 马匹倒地
   - 击中骑士: 落马
4. **战场上的箭** - 地上插满箭矢

### 骑兵冲锋

**冲锋动画**:
- 开始: 慢步
- 加速: 越来越快
- 全速: 战马飞奔
- 遇阻: 战马绊倒/骑士落马

**冲锋波次**:
- 第1-3波: 精锐骑士，秩序井然
- 第4-8波: 逐渐混乱
- 第9-12波: 混乱冲锋

### 防御工事

**坑洞效果**:
- 地面小坑（约0.5米直径，1米深）
- 隐蔽（草地覆盖）
- 效果：绊倒战马

### 相机角度

| 场景 | 相机位置 | 说明 |
|------|----------|------|
| 开场 | 高空俯视 | 展示战场全貌 |
| 英军阵型 | 侧视平视 | 长弓手列阵 |
| 法军冲锋 | 跟随骑兵 | 体验冲锋感 |
| 箭雨 | 弓手后方 | 箭矢发射视角 |
| 命中 | 低角度 | 骑士被击中 |
| 战场全景 | 高空俯视 | 战场态势 |

## 视觉特效

### 核心特效: 箭雨

1. **发射阶段**
   - 弓弦拉紧音效
   - 弓弦释放的"崩"声
   - 箭矢离弦轨迹

2. **飞行阶段**
   - 数百支箭在空中形成"箭雨云"
   - 箭矢在阳光下反射
   - 箭矢啸叫声

3. **命中阶段**
   - 击中盔甲的金属撞击声
   - 火花四溅
   - 骑士落马
   - 战马嘶鸣倒地

### 次要特效

- **尘土**: 骑兵冲锋扬起尘土
- **血迹**: 适度血迹效果
- **武器碰撞**: 金属撞击声和火花
- **旗帜**: 纹章旗在风中飘扬

### 天气效果

- 晴朗天气
- 适度云层
- 阳光从云层中透出
- 战场能见度良好

### 战场音效

- **弓弦声**: 弓弦拉紧和释放的"崩崩"声
- **箭矢声**: 箭矢飞行的"嗖嗖"声
- **金属撞击**: 箭矢击中盔甲

### 环境音效

- 风声
- 鸟鸣

- 战场的嘈杂声
