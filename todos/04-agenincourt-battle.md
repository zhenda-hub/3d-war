# 阿金库尔战役 - 实现文档

> **1415年 | 英格兰 vs 法兰西**
> 泥泞战场上的长弓屠杀，亨利五世的传奇胜利

---

## 战役信息

### 基本信息
- **时间**: 1415年10月25日
- **地点**: 法国阿金库尔（加来以南）
- **交战方**: 英格兰(约6,000人) vs 法兰西(约20,000-30,000人)
- **伤亡**: 英格兰约100-200人 vs 法兰西约6,000-10,000人
- **统帅**: 亨利五世(英) vs 德阿尔布雷特与布西考(法)

### 战术亮点
- 暴雨后的泥泞战场
- 英军选择狭窄地形迫使法军无法展开
- 长弓手近距离屠杀重骑士
- 法国贵族大量阵亡（包括3位公爵、5位伯爵）

#### 英格兰长弓手
- 身体比例: 8头身
- 装备细节:
  - 长弓(约2米)

  - 轻型皮甲

- 动画: 拉弓、放箭、装箭、移动木桩

#### 法兰西重骑士
- 身体比例: 8头身
- 装备细节:
  - 全身板甲（米兰式）- 银色/金色
  - 盔甲马 - 马身披甲

  - 骑士剑 - 腰间

- 动画: 骑马冲锋、落马、挣扎爬起

- **泥泞深度**: 膝盖深度
- **泥浆痕迹**: 战靴和马蹄的泥痕
- **泥泞粘稠**: 阻力效果

### 战斗阶段

**阶段0: 暴雨后 (0-5秒)**
- 阴沉天空，乌云密布
- 战场泥泞不堪
- 英军瑟瑟发抖（饥饿+疾病）
- 法军自信满满
- 相机展示泥泞战场

**阶段1: 法军部署 (5-15秒)**
- 法军大批骑兵集结
- 贵族骑士炫耀盔甲
- 战场狭窄，法军无法完全展开
- 相机展示法军数量优势

**阶段2: 英军挑衅 (15-20秒)**
- 亨利五世带头向法军竖起中指（历史典故）
- 长弓手整理装备
- 插入地面木桩
- 准备迎接冲锋

**阶段3: 第1-3次冲锋 (20-45秒)**
- 法军重骑士冲锋
- 英军长弓齐射
- **关键时刻**: 骑士进入泥泞地带
- **视觉效果**: 战马陷入泥沼
- **视觉效果**: 骑士落马，在泥浆中挣扎
- 箭矢穿透盔甲缝隙
- 第一波法军几乎全灭

**阶段4: 混乱冲锋 (45-70秒)**
- 后续法军被阻挡在泥泞中
- 冲锋队形混乱
- 后方部队撞上前方溃兵
- 泥泞中的骑士成为活靶子
- 英军长弓手近距离屠杀

**阶段5: 法军步兵冲锋 (70-85秒)**
- 法军步兵试图在泥泞中推进
- 同样陷入泥沼
- 英军重骑兵和步兵反击
- 白刃战

**阶段6: 战斗结束 (85-95秒)**
- 法军残部撤退
- 英军清点战利品
- 俘虏大量法国贵族
- 展示战场上的泥泞和尸体

**阶段7: 处决俘虏 (历史真相，可选)**
- 亨利五世下令处决俘虏
- 展示战争的残酷性
- 或作为历史注释提及

## 技术实现要点

### 泥泞系统（核心亮点）

**泥泞效果**:
```javascript
// 泥泞深度计算
泥泞深度 = 地点位置泥泞等级 + (单位重量 × 泥泞系数)

// 移动速度影响
移动速度 = 基础速度 × (1 - 泥泞深度 × 0.7)

// 挣扎动画
if (泥泞深度 > 膝盖深度) {
  触发挣扎动画;
  移动概率大幅降低;
}

// 泥浆飞溅
冲锋时触发泥浆粒子特效;
```

**泥泞视觉效果**:
1. **地面材质**
   - 深褐色泥浆纹理
   - 镜面反射（湿润效果）
   - 泥浆水洼

2. **泥浆痕迹**
   - 靴子踩过的泥痕
   - 马蹄印深陷
   - 泥浆飞溅轨迹

3. **单位效果**
   - 盔甲上的泥污
   - 战马身上的泥浆
   - 衣服浸透泥水

### 箭雨系统

参考克雷西战役，但增加：
- **泥泞中的骑士** - 移动缓慢，更容易命中
- **近距离射击** - 箭矢威力更大
- **密集阵型** - 法军挤在狭窄战场，命中率更高

### 骑兵冲锋+泥泞

**冲锋-泥泞交互**:
1. 开始冲锋 - 快速前进
2. 进入泥泞 - 速度急剧下降
3. 陷入泥沼 - 战马挣扎
4. 骑士落马 - 跌入泥浆
5. 挣扎爬起 - 困难重重
6. 成为靶子 - 被箭矢射杀

### 狭窄战场

**地形限制**:
- 战场宽度仅750-1000米
- 两侧是森林（不可通行）
- 法军无法展开优势兵力
- 法军拥挤在自己后方单位上

## 视觉特效

### 核心特效: 泥泞战场

1. **泥泞地面**
   - 深褐色泥浆
   - 镜面反射
   - 泥浆水洼

2. **泥浆飞溅**
   - 骑兵冲锋时泥浆四溅
   - 泥浆粒子系统
   - 泥浆飞溅轨迹

3. **泥泞中的挣扎**
   - 战马陷入泥沼
   - 骑士跌落泥浆
   - 挣扎爬起动画
   - 泥浆中的绝望

### 次要特效

- **箭雨**: 类似克雷西
- **尘土**: 较少（因为泥泞）
- **血迹**: 混合泥浆，变成暗红色
- **天气**: 阴沉，乌云，小雨（可选）

### 天气效果

- **暴雨后**: 乌云密布
- **阴沉光线**: 灰暗色调
- **小雨**: 营造悲惨氛围
- **雾气**: 战场远处模糊
