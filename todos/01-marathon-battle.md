# 马拉松战役 - 实现文档

> **公元前490年 | 雅典 vs 波斯帝国**
> 方阵战术击败波斯大军，保卫西方文明

---

## 战役信息

### 基本信息
- **时间**: 公元前490年9月
- **地点**: 马拉松平原（希腊阿提卡半岛）
- **交战方**: 雅典(约11,000人) vs 波斯帝国(约20,000-25,000人)
- **伤亡**: 雅典192人 vs 波斯6,400人
- **统帅**: 米太亚德(雅典) vs 达提斯(波斯)

### 战术亮点
- 希腊重装步兵方阵冲锋战术
- 故意加强两翼，中央薄弱
- 双翼合围波斯军队的经典战术

## 3D场景设计

### 地形

```
┌─────────────────────────────────────────┐
│  雅典阵地    ← 战场宽度约1500米 →     │
│  ┌──────┐                              │
│  │ 方阵  │    ┌─────────────────┐    │
│  │ 阵地  │    │   平原战场      │    │
│  └──────┘    └─────────────────┘    │
│              ┌───────────┐            │
│              │  沼泽地   │            │
│              └───────────┘            │
│              [波斯营地]               │
└─────────────────────────────────────────┘
```

**地形要素**:
- 马拉松平原（开阔草地/沙地）
- 西南侧沼泽地（不可通行区域）
- 简单起伏地形
- 爱琴海远景

### 兵种设计

#### 雅典军
| 兵种 | 特征 | 武器 | 装备 |
|------|------|------|------|
| 重装步兵 | Hoplites | 短剑(约0.6m)、长矛(约2.5m) | 圆盾(约0.9m)、青铜头盔、胸甲 |

#### 波斯军
| 兵种 | 特征 | 武器 | 装备 |
|------|------|------|------|
| 轻步兵/弓箭手 | 移动快 | 短弓(约1.2m)、短矛 | 轻型护具、布帽 |

### 单位模型

#### 雅典重装步兵
- 身体比例: 8头身
- 装备细节:
  - 圆盾(约0.9m直径) - 带希腊图案
  - 青铜头盔 - 马鬃冠饰
  - 白色/米色长袍
  - 褐色凉鞋
- 动画: 持矛前进、举盾、冲锋

#### 波斯弓箭手
- 身体比例: 8头身
- 装备细节:
  - 弓箭 - 复合弓
  - 箭袋 - 腰间悬挂
  - 彩色长袍 - 蓝色/红色
  - 软帽/头巾
- 动画: 拉弓、放箭、逃跑

### 战斗阶段

**阶段1: 冲锋 (0-10秒)**
- 雅典方阵开始冲锋
- 波斯弓箭手开始射击
- 箭矢飞向雅典方阵

**阶段2: 接触 (10-20秒)**
- 雅典中央故意后退
- 两翼保持推进速度
- 波斯中央突破雅典中央

**阶段3: 包围 (20-40秒)**
- 雅典两翼转向
- 形成包围圈
- 波斯军队被包围

**阶段4: 追击 (40-50秒)**
- 波斯军队溃逃
- 雅典军追击
- 部分波斯士兵逃向海上船只

## 技术实现要点

### 相机角度

| 场景 | 相机位置 | 说明 |
|------|----------|------|
| 开场 | 侧后方高空 | 展示双方阵型 |
| 冲锋 | 中距离跟随 | 跟随方阵冲锋 |
| 接触 | 低角度近景 | 感受冲锋气势 |
| 包围 | 俯视全景 | 展示包围战术 |
| 追击 | 混合剪辑 | 混乱战斗场面 |

### 视觉特效

1. **箭雨效果**
   - 大量箭矢抛物线飞行
   - 箭矢击中盾牌的金属音效
   - 部分箭矢插在盾牌上

2. **尘土效果**
   - 方阵冲锋扬起的尘土
   - 战场氛围营造

3. **光照**
   - 地中海阳光
   - 阴影效果
   - 战场烟尘

### 音效设计

- 战鼓声 - 节奏感
- 冲锋呐喊 - 希腊语/波斯语
- 武器碰撞声
- 箭矢飞行声
- 胜利欢呼
